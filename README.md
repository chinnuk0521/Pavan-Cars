## The WhatsApp Booking Engine

Zero backend infrastructure. Instant confirmations. The booking flow is elegant by design.

<br/>

```
  01  ·  Customer completes the structured booking form
           ↓
  02  ·  JavaScript validates all fields + 10-digit phone format
           ↓
  03  ·  A complete, structured booking message is assembled
           ↓
  04  ·  WhatsApp opens — message pre-filled, ready to send
           ↓
  05  ·  Proprietor receives full booking details instantly
```

<br/>

**Fields captured:** Guest Name · Contact Number · Pickup Address · Travel Date · Vehicle Class · Concierge Notes

> No missed bookings. No follow-up friction. The customer confirms in one tap.

## Customisation Reference

<br/>

### — Updating Contact Numbers

Two numbers power the platform: `9908966682` (Direct Line) and `9381286787` (WhatsApp Concierge).

**In `index.html`** — update both the display text and the `href`:

```html
<a href="tel:9908966682">+91 9908966682</a>
```

**In `script.js`** — update the WhatsApp booking destination:

```javascript
// Locate within the form submission handler
const whatsappNumber = '919908966682';

// Format: country code + number, no spaces or dashes
const whatsappNumber = '91YOURNUMBER';
```

<br/>

### — Replacing Vehicle Images

**Fleet cards** in `index.html`:

```html
<img src="YOUR_IMAGE_URL" alt="Vehicle class name" />
```

**Hero background** in `style.css`:

```css
.hero {
  background-image: url('YOUR_HERO_IMAGE_URL');
}
```

Supports absolute URLs (Unsplash, Cloudinary) or relative local paths.

<br/>

### — Updating Copy & Taglines

All visible text is in `index.html`, structured and clearly commented:

```html
<!-- Hero Section -->
<p class="subtext">Your updated tagline here</p>
```
